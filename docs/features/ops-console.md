# Ops Console

The llm.port Ops Console provides operational control for platform administrators.

## Core capabilities

- Service lifecycle operations
- Deployment status visibility
- Runtime and system health checks
- Log and troubleshooting entry points

## Governance value

- Centralized operator workflow
- Reduced reliance on ad-hoc host access
- Better auditability of operational actions

## Best-practice usage

- Restrict ops permissions to designated operators
- Use change windows for production-impacting actions
- Keep a documented rollback process
