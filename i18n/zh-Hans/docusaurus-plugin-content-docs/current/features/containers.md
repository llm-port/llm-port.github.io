# 运维控制台

管理前端的**运维控制台**提供完整的容器和基础设施管理。

## 容器管理

- **生命周期控制**：启动、停止、重启、暂停、恢复、删除
- **Exec**：通过管理 UI 在容器内运行命令
- **日志流**：带跟随模式的实时容器日志
- **健康检查**：自动监控的容器健康状态

## 镜像管理

- **带 SSE 进度的拉取**：带实时进度流的镜像拉取
- **列出与检查**：查看所有镜像的大小、标签和创建日期
- **清理**：删除未使用的镜像以回收磁盘空间

## Compose Stack 管理

- **部署 / 更新**：验证后应用 Compose 配置
- **回滚**：通过差异视图回滚到先前版本
- **版本追踪**：每次部署创建带审计追踪的编号版本
- **差异视图**：比较版本间的配置差异

## 容器分类

容器按管理和策略分类：

| 类别          | 描述         | 示例                       |
| ------------- | ------------ | -------------------------- |
| `SYSTEM_CORE` | 核心平台服务 | Backend、Gateway、Frontend |
| `SYSTEM_AUX`  | 支撑基础设施 | Grafana、Loki、Langfuse    |
| `LLM_ENGINE`  | 模型运行时   | vLLM、Ollama、llama.cpp    |
| `TENANT_APP`  | 用户应用     | 自定义容器                 |

每个类别可以有自己的管理策略（重启策略、资源限制等）。
